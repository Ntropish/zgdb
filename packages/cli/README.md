# @zgdb/cli

This package contains the command-line interface (CLI) for `zgdb`.

`zgdb` is a synchronous, in-memory graph database client generated from Zod schemas. You use this CLI tool to process your schemas and generate the necessary database and client code.

### Documentation

For full documentation on usage and examples, please see the main project README on our [GitHub homepage](https://github.com/ntropish/zgdb).
